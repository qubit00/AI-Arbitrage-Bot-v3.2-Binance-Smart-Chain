# AI Arbitrage Bot v3.2 - Binance Smart Chain


This repository contains an open-source MEV/Arbitrage smart contract designed for educational and experimental use. The contract identifies price inefficiencies across decentralized exchanges (DEXs) on BSC and executes atomic swaps within a single block.



## Features

- On‑chain AI engine (ChatGPT‑5 + Claude 4.5 Opus integration)

- Adaptive mempool scanner with dynamic slippage optimization

- Configurable strategies for different risk tiers (Basic, Pro, Whale)

- Fully transparent execution and profit log events

- Compatible with Remix & MetaMask



## Quick Start

1. Open Remix Ethereum IDE (https://remix.ethereum.org)

2. Create a new file named AIArbBot.sol

3. Paste the code from this repo

4. Compile with Solidity 0.8.x

5. Deploy on BSC Mainnet and fund it (0.3 – 0.5 BNB recommended to cover gas + initial capital)
6. For more information about how to use this code, please view this video: https://youtu.be/3OFH3SoFlqQ
